# 站点搭建过程

基于 hexo 搭建了自己的站点，在此发布一下搭建教程。
<!--more-->

## 依赖

+ 配置 node.js
+ 配置 hexo


## 定制
+ 设置 theme


## 命令行操作

1. `hexo new <title>`

   创建新文章
   
2. `hexo g`

   构建静态页面(`hexo generate` )

3. `hexo clean`

   清除已构建的页面

4. `hexo s`

   本地服务器(`hexo server`)

5. `hexo d`

   发布、部署(`hexo deploy`)
